# Aurora

**Aurora** is a lightweight, cross-platform music player built for audiophiles and casual listeners alike. It delivers a clean, intuitive interface
and broad audio format support â€” including high-quality lossless formats.

![Main UI](docs/images/screenshot-main-ui.png)

## Features

- Select a directory to add tracks
- View music by album and artist
- Playlist management: create, edit, and manage playlists locally
- Manage playback queue
- Clean and simple user interface
- Works across platforms (Windows, macOS, Linux)

## Development

Pre-requirements:

- NodeJS - `^20.19.5`
- Yarn - `^1.22.22`

Install packages:

```bash
yarn install
```

Start the app in `dev` environment:

```bash
yarn start
```

## Packaging

To package app for local platform:

```bash
yarn package
```
